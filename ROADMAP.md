# ClawNet 路线图

## 阶段 0：概念验证（现在 — 2026年6月）

**目标：两台设备组网，跑通全链路。**

- [ ] 节点发现：局域网内 gossip 握手
- [ ] CRDT 令牌台账：两个节点共享并自动合并
- [ ] 任务分派：节点A发布任务 → 节点B认领 → 执行 → 回传 → 结算
- [ ] worker-dom：浏览器DOM操作（OpenClaw 集成）
- [ ] CLI 工具：`clawnet start` / `status` / `tokens` / `task submit`

**交付物：** 可跑 demo（两台笔记本组网，A让B帮忙搜B站 → B回传结果给A）

---

## 阶段 1：MVP（2026年7月 — 8月）

**目标：10+ 节点，跨公网组网，支持批量搜索/精炼任务。**

- [ ] NAT 穿越（libp2p / STUN）
- [ ] 任务市场：按令牌余额排序的自动调度
- [ ] 信任积分：完成任务 → 加分；失败 → 扣分
- [ ] worker 模块热插拔
- [ ] 公网种子节点（由社区志愿者维护）
- [ ] 基础 Web 面板（可选）

**交付物：** 社区部署 10+ 节点，可跑真实任务（搜索精炼、批量摘要）

---

## 阶段 2：扩展（2026年9月 — 12月）

**目标：100+ 节点，支持 AI 图片生成。**

- [ ] worker-image-gen：调用本地 GPU/神经引擎做图片推理
- [ ] 模型分片分发（IPFS/BT）
- [ ] 任务冗余执行（同任务派多个节点 → 取最先返回的 K 个）
- [ ] 结果验证（C2C 抽检）
- [ ] Web 面板完善
- [ ] Docker/Homebrew 一键安装

**交付物：** 社区可跑"夜间批量生图"任务

---

## 阶段 3：视频生成（2027年）

**目标：支持分布式视频渲染。**

- [ ] 骨架生成协议（发起者本地跑轻量潜空间定位）
- [ ] 帧级别并行渲染（贡献者认领帧区间）
- [ ] 重叠帧 + 渐变拼接
- [ ] 长视频支持（分段 → 并行 → 合并）
- [ ] 移动端支持（iOS/Android 节点）

**交付物：** 社区可跑"睡前提交视频prompt，早上收成片"

---

## 阶段 4：生态（2027+）

**目标：ClawNet 成为分布式算力的标准协议。**

- [ ] 联邦微调
- [ ] 物理模拟（FEM/CFD 网格分发）
- [ ] 嵌入式设备支持（树莓派/路由器）
- [ ] 非 AI 任务（转码/渲染/编译）
- [ ] 插件市场（第三方 worker 模块）

---

## 不做什么

- 不发行代币。ClawNet 的令牌不是货币，是排位凭证
- 不做智能合约。不需要共识机制
- 不做KYC。不需要实名，不需要注册
- 不承诺"跑任务赚生活费"。这是互助网络，不是副业平台
