# ClawNet

**C端去中心化算力网络 — 把你闲置的设备变成全球算力的一部分**

ClawNet 是一个开源的、完全去中心化的算力共享网络。你可以把自己的电脑、手机、平板的闲置算力贡献出来跑AI任务（批量视频生成、大型推理、物理模拟等），同时用别人贡献的算力跑你自己的任务。

**不买卡、不建数据中心、不需要法人资格。** 一台设备就能入网。

---

## 为什么做 ClawNet

AI 的算力正在被少数公司垄断。训练一个前沿模型的硬件成本已经超过1亿美元，数据中心电费比小国 GDP 还高。这不对。

但用户手里有的是算力——全球20亿部智能手机的神经引擎，加起来每年以数亿台新设备的速度在升级。这些算力99%的时间在待机。

ClawNet 就是把这些闲置算力织成一张网。**不靠资本堆砌，靠协议协作。**

## 快速开始

### 加入网络

```bash
# 安装 ClawNet 节点
curl -sSL https://clawnet.dev/install.sh | bash

# 启动节点（会自动发现局域网/公网其他节点）
clawnet start

# 查看节点状态
clawnet status

# 查看令牌
clawnet tokens
```

### 提交任务

```bash
# 让网络帮你搜索并精炼信息（需要先积累令牌）
clawnet task submit --type batch-search --query "华为畅享90 测评" --sources "bilibili,douyin"

# 生成视频
clawnet task submit --type video-gen --prompt "一个穿红色风衣的人走在雨夜东京街头" --duration 30s
```

## 核心架构

ClawNet 的核心是三层协议：

```
┌──────────────────────────────────────────────────┐
│                  任务公告板                         │
│          (Gossip DHT, 无中央数据库)                  │
├──────────────────────────────────────────────────┤
│ ┌──────────┐  ┌──────────┐  ┌──────────┐        │
│ │ 节点 A   │  │ 节点 B   │  │ 节点 C   │  ...    │
│ │ 手机     │  │ 笔记本   │  │ 旧平板   │         │
│ └──────────┘  └──────────┘  └──────────┘        │
│      │              │              │              │
│ ┌────┴────┐  ┌────┴────┐  ┌────┴────┐          │
│ │调度器   │  │调度器   │  │调度器   │  ← 每个人    │
│ │+ 工人   │  │+ 工人   │  │+ 工人   │    都是对等   │
│ └─────────┘  └─────────┘  └─────────┘          │
├──────────────────────────────────────────────────┤
│             令牌台账 (CRDT, 无区块链)                │
└──────────────────────────────────────────────────┘
```

## 核心特性

### 1️⃣ 纯对等网络
没有中央服务器、不是区块链挖矿、不需要KYC。每个节点在网内拥有相同的权利。

### 2️⃣ 令牌经济
- 贡献算力 → 获得令牌
- 消耗令牌 → 提交任务优先调度
- 令牌 = 网络优先使用权，不是货币

### 3️⃣ 任务模块化
任务类型可插拔。目前支持：
- ✅ DOM检索（爬取检索替代API爬虫）
- 🚧 批量AI图片生成
- 🚧 批量视频生成（去中心化帧渲染）
- 🚧 联邦微调
- 🚧 物理模拟（FEM/CFD）

### 4️⃣ 隐私优先
- 数据不经过中央服务器
- 任务直发直收
- 本地推理不上云（模型在节点本地加载）

## 技术栈

| 组件 | 选择 | 理由 |
|---|---|---|
| **节点发现** | SWIM / Memberlist | P2P gossip，百万级验证 |
| **令牌台账** | CRDT (Automerge/Yrs) | 无冲突合并，不需要共识 |
| **任务传输** | libp2p / 直连 | 去中心化，NAT穿透 |
| **模型分片** | IPFS + BitTorrent | 海量节点分发验证过 |
| **设备端推理** | WebGPU / CoreML / ONNX | 跨平台，不依赖云端 |
| **客户端** | 轻量 CLI + 可选 GUI | 门槛低 |
