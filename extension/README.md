# ClawNet 浏览器插件

把你的闲置浏览器算力变成全球 AI 网络的一部分。

## 安装（开发者模式）

由于尚未上架 Chrome Web Store，目前需要通过开发者模式加载：

1. 打开 Chrome 浏览器，访问 `chrome://extensions`
2. 开启右上角的 **「开发者模式」**
3. 点击 **「加载已解压的扩展程序」**
4. 选择本目录（`extension/`）

加载成功后，工具栏会出现 ClawNet 图标。

## 功能

### 仪表盘
- 节点状态（在线/离线/计算中）
- 令牌余额
- 在线节点数
- 设备能力（WebGPU、DOM 等）
- 自动接单开关

### 提交任务
- **网页搜索** — 让远程节点替你搜索页面内容
- **提取数据** — 从页面提取结构化信息
- **精炼摘要** — AI 自动总结页面内容
- **监控变化** — 监控页面指定元素的变化
- **图片生成** — 通过 WebGPU 生图（需支持 WebGPU 的设备）

### 后台运行
插件安装后自动在后台运行：
- 连接信令服务器发现其他节点
- 通过 WebRTC 与其他节点 P2P 通信
- 自动接单执行任务（如果开启）
- 定时同步令牌台账

## 构建

```bash
# 生成 PNG 图标 + 打包
node scripts/build.js
```

## 架构

```
background (service worker)
  ├── WebRTC → 与其他节点 P2P 通信
  ├── WebGPU → 本地推理加速
  └── Message → 与 popup / content script 通信
        │
        ▼
popup (用户操作界面)
  ├── 状态面板
  ├── 任务提交
  └── 日志

content script (注入到每个页面)
  ├── 搜索页面内容
  ├── 提取结构化数据
  ├── 填写表单
  └── 监控页面变化
```
